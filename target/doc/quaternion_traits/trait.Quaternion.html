pub trait Quaternion<Num: Axis> {
    // Required methods
    fn r(&self) -> Num;
    fn i(&self) -> Num;
    fn j(&self) -> Num;
    fn k(&self) -> Num;
}
Expand description

The general representation of any quaternion type.

Note: The r, i, j and k methods are used as if they are cheap operations.
